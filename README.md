# hololiveEN-image-classify
A image classification for hololiveEN characters

A demo of files of training model to identify characters in images.

How to run:
(1). install cuda, cudnn, zilb
(2). install python package by using pip(or conda) and requirments.txt

note: 
(1).You have to find images to train/test by youself.
(2).You have to change some path/configuration in code by yourself.
